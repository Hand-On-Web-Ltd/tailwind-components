# Tailwind CSS Business Components 🎨

A collection of free, copy-paste Tailwind CSS components built for business websites. Each component is a standalone HTML file — just grab the code you need.

## Components

| Component | Description |
|-----------|-------------|
| [Hero Section](components/hero-section.html) | Full-width hero with headline, subtitle, and CTA buttons |
| [Pricing Table](components/pricing-table.html) | Three-tier pricing with a highlighted "popular" plan |
| [Testimonials](components/testimonial-section.html) | Customer testimonial cards with photos and quotes |
| [Contact Form](components/contact-form.html) | Clean contact form with validation styling |
| [Footer](components/footer.html) | Multi-column footer with links and social icons |
| [Feature Grid](components/feature-grid.html) | Icon-based feature grid for services or benefits |

## How to Use

1. Open any component HTML file
2. Copy the HTML you need
3. Paste it into your project

Each file includes the Tailwind CDN, so you can preview them directly in your browser.

If your project already uses Tailwind, just copy the inner HTML — skip the CDN script tag.

## Preview

Open `index.html` to see all components in one page.

## Tech

- Tailwind CSS via CDN (no build step needed)
- Pure HTML — no JavaScript required
- Mobile responsive
